# Technical Demo for C++

This is techincal demo for C++ written by Jacky Cheung from scatch.

# Dependency

GNU C++ v6.3 or above [https://gcc.gnu.org/]

Tup build system [http://gittup.org/tup/index.html]

# Build

Build by script.
```
.\build.sh
```

Or, use Tup to build.
```
tup init
tup
```

# Run Test

```
.\build\test
```
